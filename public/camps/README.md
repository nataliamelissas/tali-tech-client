## Image Guidelines

- Format: JPG or PNG
- Resolution: Optimized for web (aim for under 500KB each)
- Orientation: Landscape preferred (800x600 or similar)
- Content: Real, candid photos showing engagement and learning
- Privacy: Ensure you have parent permission to use student photos
