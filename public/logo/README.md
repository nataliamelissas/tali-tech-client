# Logo Directory

## Required Logo File

- **talitech-fox-logo.png** (200x200 px)
  - Format: PNG with transparent background
  - TaliTech fox mascot logo
  - Used in Navbar and footer

## Usage

Import logo in components:
```typescript
import Image from 'next/image';

<Image 
  src="/logo/talitech-fox-logo.png" 
  alt="TaliTech Logo" 
  width={200} 
  height={200} 
/>
```
