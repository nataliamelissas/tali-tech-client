# Stock Photos Directory

## Required Stock Photos

These should be professional stock photos showing kids coding/learning:

1. **hero-kids-coding.jpg** (1920x1080 px)
   - Homepage hero section
   - Diverse group of kids engaged with computers
   - Bright, inviting, professional quality

2. **scratch-programming.jpg** (800x600 px)
   - Camp page - Week 1 header
   - Kids using Scratch or block-based coding
   
3. **minecraft-education.jpg** (800x600 px)
   - Camp page - Week 2 header
   - Students learning with Minecraft Education

4. **python-ai-learning.jpg** (800x600 px)
   - Camp page - Week 3 header
   - Older students coding in Python or AI projects

5. **diverse-students.jpg** (800x600 px)
   - About page header
   - Diverse group of students collaborating

## Image Guidelines

- Format: JPG or PNG
- Resolution: High quality for web (optimized but not pixelated)
- Style: Bright, colorful, engaging
- Subjects: Diverse group of children, focus on engagement/learning
- Mood: Fun, educational, inclusive

## Recommended Stock Photo Sources

- Unsplash (free)
- Pexels (free)
- iStock (paid)
- Getty Images (paid)

Search terms: "kids coding", "children programming", "STEM education", "diverse students learning"
